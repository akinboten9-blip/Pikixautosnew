# **App Name**: PIKIX AUTOS

## Core Features:

- Vehicle Search & Filtering: Advanced search capabilities including make, model, year, price range, mileage, and location.
- AI Recommendations: AI-powered recommendations engine that suggests vehicles based on user preferences and browsing history.
- Vehicle Listing Management: Dashboard for dealers/admin to manage vehicle inventory, including bulk uploads and status tracking.
- Payment Integration: Secure payment processing via Flutterwave/Paystack with options for installment payments and escrow services.
- Virtual Vehicle Tours: Interactive virtual tours with 360Â° images/videos to showcase vehicle features.
- Finance Calculator: Embedded finance calculator tool for estimating loan payments.
- Shipping & Logistics Integration: Calculate port clearance, offer domestic delivery, real-time tracking and estimate custom duties.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey trust and sophistication in the automotive sector.
- Background color: Light grey (#F0F0F3) to provide a clean, neutral backdrop that highlights vehicle listings.
- Accent color: Teal (#009688) for interactive elements and calls-to-action to add a vibrant, modern touch.
- Body and headline font: 'Inter', a sans-serif font for clear and modern readability.
- Consistent, modern line icons for navigation and key features, enhancing UX.
- Mobile-first responsive design with a clean, intuitive layout.
- Subtle transition animations and loading indicators to improve perceived performance.
